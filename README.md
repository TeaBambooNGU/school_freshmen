# 程序功能：

1. 新生的个人信息登记
2. 校园的介绍
3. 校区的导航（只是一个导航二维码，哈哈）
4. 校区游览（用Unity3D做的一个可以游览校区的小游戏，因为不会建模，模型太丑就不搬上来了）
5. 周边美食-- 系统自动排名，出最好的9家，然后你可以在程序里点击按钮，进入商户页面直接下单，商户数据来源于饿了嘛(本来打算爬美团的，但是他封了我的IP。。。）
